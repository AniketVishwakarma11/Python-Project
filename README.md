# Capstone-Project
Project Overview

This capstone project analyzes online movie ratings to investigate whether Fandango’s movie ratings in 2015 were biased toward rating movies higher than they should have been. The project is inspired by the FiveThirtyEight article that questioned the trustworthiness of Fandango’s ratings, especially because the platform also makes money by selling movie tickets.

The main goal of this project is to use Python, Pandas, and data visualization techniques to compare Fandango ratings with ratings from other movie platforms and determine whether there was evidence of rating inflation.

The objective of this project is to:

Analyze movie ratings data from Fandango and other rating platforms
Identify whether Fandango’s displayed ratings were consistently higher
Compare rating distributions and patterns using Python
Use data cleaning, exploratory data analysis (EDA), and visualization to draw meaningful conclusions
Validate whether the findings support the claims made in the FiveThirtyEight article

Tools & Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn 
Jupyter Notebook

Key Insights

Some of the major insights from this project include:

Fandango ratings appeared to be skewed toward higher values
Rating distributions suggested that many movies were rated more positively on Fandango than on other platforms
The analysis supported the idea that Fandango’s displayed ratings may have been inflated
Visual comparisons made it easier to identify patterns and bias in the rating system

Final conclusion: The project suggests that Fandango’s 2015 movie ratings showed signs of bias toward higher ratings, which aligns with the concerns raised in the FiveThirtyEight article.
